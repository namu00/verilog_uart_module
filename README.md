# verilog_uart_module
> ### Variable baudrate support uart module
## Specs
|Object|Number|  
|:--:|:--:|  
|CLOCK FREQ| 50MHz or Above|  
|RESET TYPE| neg-active reset|
|BAUD RATE| Variable (2400 ~ 250K)|  
|CONNECTOR TYPE| RS-232 or CP210x|  
|I/O BUFFER SIZE|8 Bit|   
|STOP BIT|1 Bit|  
|PARITY BIT|NO|
